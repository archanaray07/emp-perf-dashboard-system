# Employee Performance Dashboard

A full-stack Employee Performance Dashboard built using Django, React, and MySQL.

## Tech Stack

### Backend
- Python
- Django
- Django REST Framework
- MySQL

### Frontend
- ReactJS
- Axios
- Bootstrap / CSS

## Features
- Employee management
- Performance tracking
- Authentication system
- REST APIs
- Dashboard analytics

## Project Status
Frontend development is currently in progress.

### Backend Setup
